# GNotifier

Add-on for Firefox and Thunderbird. Replaces built-in notifications with the OS native notifications. It supports most Linux Desktop Environments, Windows 8.1 & 10.

## Build instruction

Make sure that you have installed [jpm](https://developer.mozilla.org/en-US/Add-ons/SDK/Tools/jpm#Installation) tool.

1. Open a terminal in the ./source dir
2. Run ``jpm xpi`` to make the xpi
